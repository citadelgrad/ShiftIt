ShiftIt version 2.0.0

Major Release - This is a significant update with modernized architecture and new features.

What's New

  • Sparkle 2.8 Integration - Updated auto-update framework with improved security (EdDSA signatures)
  • Swift 6 Modernization - Modern window management using Swift 6 with Sendable conformance
  • New Keyboard Shortcut Recorder - Replaced deprecated ShortcutRecorder with modern KeyboardShortcutRecorder
  • Ultrawide Monitor Support - New actions to divide screen by 6 or 3 for ultrawide displays
  • Improved Accessibility - Fixed permissions dialog and accessibility handling
  • Removed X11 Support - Deprecated X11WindowDriver has been removed

Requirements

  • macOS 14.6 (Sonoma) or later - This version requires a modern macOS release

More information about this release can be found on GitHub.


If you find any bugs please report them on GitHub Issues.
